# Products.WebApi.Spa

This is the api for SPA

## To run you need

- Docker
- Docker Compose


## How to run

In the project root

Start the database
```
docker-compose up
```

And start the application
```
dotnet run
```
Then Will Open the Swagger 

And start the sync API
[Sync.WebApi](https://github.com/anologicon/Sync.WebApi)
